Add to the weekly scout source list immediately and subscribe to releases. Any plugin worth adopting will show up here before it shows up anywhere else.

**2. `anthropics/skills` — `skill-creator` (24/25) — USE AS-IS**
The official skills repo now ships a `skill-creator` skill (and other example skills) installable via `/plugin install example-skills@anthropic-agent-skills`. The `skill-creator` is directly useful for the T4.3 workflow and complements the scout agent — scout finds candidates, `skill-creator` builds them. Install now: it's Apache 2.0, well-documented, and one command to wire in.
